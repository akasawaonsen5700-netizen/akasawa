ナレーション音声は Cartesia.ai のクローンボイス生成APIを使用します。

① 投稿原案の作成 (台本入力)

入力されたテキストを元に、自動で遠藤正俊氏のクローン音声およびテロップ付き縦型リール動画（Instagram用）と、X用の投稿文章が生成されます。
動画の背景として使用する画像や動画を選択してください。未選択の場合はデフォルト背景画像が使用されます。
遠藤正俊氏自身の録音音声（endo.mp3など）を直接使用したい場合は選択してください。未選択の場合は、Cartesia.ai APIを用いて本人のクローン音声が自動合成されます。

② 配信フローとシステム連携

Web画面からの登録を契機に、Gemini API を用いた「情緒的なInstagramキャプション/ナレーション台本」および「X用の思考を促す短文」の下書きが自動生成されます。

Cartesia.ai API ＆ Remotion 自動レンダリング連携中
登録と同時にバックグラウンドで自動的に遠藤正俊氏のクローン音声が生成され、テロップとBGMが一体となった本物の縦型動画（MP4）が自動で書き出されます。
  • AI下書き生成: 遠藤氏の戦略PDFを学習したプロンプトでGeminiが自律的に起稿
  • クローン音声合成: Cartesia.aiのボイスクローニングAPIと連携し、遠藤正俊氏本人の声で台本を朗読する音声を自動生成
  • 自動ビデオ書き出し: 登録直後に裏で Remotion レンダリングを起動し、テロップ・BGM・クローン音声を合体させたMP4動画を自動生成
  • プレビュー＆直接承認: 承認キューで完成したMP4動画をプレビューし、モーダル内からワンクリックで直接承認可能
  • 自律スケジュール配信: 承認済みの投稿はスケジュールされた日時に自動で各SNS（Instagram / X）へWebhook配信
